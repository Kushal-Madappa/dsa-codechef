# dsa-codechef

My CodeChef solution log. Solutions are written in **Python 3** (with C++
added when time limits demand it). Focus areas: competitive-programming
patterns, fast I/O, ad-hoc problem solving, and contest preparation.

## Structure

```
dsa-codechef/
├── beginner/     # 1-star / starter problems
├── easy/         # 2-star problems
├── medium/       # 3-star problems
└── contests/     # Long Challenge / Cook-Off / Lunchtime submissions
```

## Solution log

| Code | Problem | Difficulty | Topic(s) | File |
|------|---------|------------|----------|------|
| INTEST | Enormous Input Test | Beginner | Fast I/O | [beginner/INTEST_enormous_input_test.py](beginner/INTEST_enormous_input_test.py) |
| HS08TEST | ATM | Beginner | Conditionals | [beginner/HS08TEST_atm.py](beginner/HS08TEST_atm.py) |

## Running

```bash
echo "3 2
1
2
3" | python beginner/INTEST_enormous_input_test.py
# expected: 1   (only 2 is divisible by 2)

echo "30 120.00" | python beginner/HS08TEST_atm.py
# expected: 89.50
```
